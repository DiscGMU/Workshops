QGIS Workshops

Workshops by Jack Tieszen

  • Intro to QGIS

    • An introduction to the open-source platform known as QGIS. Learn how to navigate the interface, download plugins, gather data from Open Street Map, edit attributes, import data, create a choropleth map, export the map.

  • QGIS Intermediate

    • In a continuation of the QGIS series this workshop will teach you how to accomplish a nearest neighbor analysis and how to calculate the area of a raster image.

  • Create a Dot Density Map in QGIS

    • In this workshop learn how to create a dot density map with census data.

  • Python in QGIS: PyQGIS

    • In this workshop learn the basics of using the programming language Python in QGIS. Learn to input data and use the toolbox with just the Python window.

  • Georeferencing in QGIS

    • Learn how to take an image and overlay it on a dymanic map in QGIS.
